# Release notes — July 2026

## Theme & Design

- **5 color themes**: light, dark, forest, ocean, sepia — toggle cycles through all five
- **Theme-aware components**: primary buttons, theme toggle, work cards, memory strips all follow active theme
- **Impeccable audit**: reduced side-tab accent stripes 3-5px → 2px, removed layout-thrashing padding transitions. Both demo and template pass with 0 anti-patterns
- **Contrast fixes**: accent-on-accent ratios meet WCAG AA (dark 3.1:1, sepia 4.5:1)
- **Reduced motion + forced colors**: `prefers-reduced-motion` zeroes animations; Windows High Contrast Mode gets visible borders
- **Form polish**: input padding tightened 13→8px, spotlight cards get 16px bottom margin, date fields responsive with `min-width: 0`

## UX improvements

- **Friendlier labels**: pack detail buttons say "Open work path", "Clear blocker", "Pick what runs next" instead of "Start", "Unblock", etc.
- **Purpose display**: pack detail shows `purpose` field when set
- **Contextual guidance**: each pack detail tells you exactly what step comes next
- **Hover help tips** (?): What to do, Work path, Blocker, Next action, Proof target — all tooltipped
- **Seed titles**: kebab-case slugs (`lighting-checklist`) → proper English ("Lighting checklist")
- **"Do" replaces "run"**: all user-facing text now says "do next" instead of "run next"
- **Confetti burst**: emoji rain (🎉✨🎊✅🌟) on Finish with proof

## Table/compact view
